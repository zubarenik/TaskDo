# Итоговый проект группы МШП 303

### Примерная функциональность проекта

Проект представляет собой _task_-менеджер с элементами органайзера. Особенностью системы является наличие онлайн-игры, которую можно проходить не отрываясь от работы.


### Запуск Pylint

`pylint TaskDo/**/*.py-для проверки проекта`

### Для тех у кого проблемы с установкой channels
```console
sudo apt-get install python3-dev
```
